Esse é o HTML puro, sem o css e sem o javascript.

ao clicar nesse botão, ira se ativar o css usando uma função javascript.

Explicação:
Ao clicar no botão, o JavaScript cria dinamicamente uma tag <link> que referencia o arquivo style.css e adiciona essa tag no <head> do HTML. Assim, os estilos definidos no CSS passam a ser aplicados na página instantaneamente.
